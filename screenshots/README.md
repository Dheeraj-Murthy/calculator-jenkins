# Screenshots Directory

This directory contains screenshots for the project report.

## Required Screenshots

1. **dockerhub-repo.png** - DockerHub repository page showing your calculator image
2. **jenkins-build-success.png** - Jenkins pipeline showing successful build completion
3. **jenkins-pipeline-stages.png** - Detailed view of all pipeline stages
4. **test-results.png** - JUnit test results in Jenkins
5. **docker-image-details.png** - DockerHub image details and tags

## How to Add Screenshots

1. Take screenshots of your DockerHub repository and Jenkins builds
2. Save them in this directory with the names listed above
3. Update the PROJECT_REPORT.md file if you use different filenames
4. Make sure screenshots are in PNG format for best quality

## Screenshot Guidelines

- **DockerHub**: Show the repository page with image tags, pull count, and description
- **Jenkins**: Show the complete pipeline with all green (successful) stages
- **Test Results**: Show the JUnit test summary with pass/fail counts
- **Pipeline Stages**: Show the detailed view of each stage execution
- **Image Details**: Show DockerHub image layers, size, and metadata
